# Документация CryptoMath Library

## Онлайн документация

Документация автоматически генерируется и публикуется на GitHub Pages при каждом коммите в ветку `main` или `master`.

**Ссылка на онлайн документацию:** См. README.md в корне проекта

## Локальная генерация документации

Для локальной генерации документации используйте Doxygen:

```bash
doxygen Doxyfile
```

Документация будет создана в директории `docs/html/`.

## Просмотр документации

Откройте файл `docs/html/index.html` в браузере:

```bash
# Linux
xdg-open docs/html/index.html

# Или просто
firefox docs/html/index.html
# или
google-chrome docs/html/index.html
```

## Что включено в документацию

- Все классы и структуры
- Все функции и методы
- Концепты C++20
- Математические формулы (через MathJax)
- Иерархия классов
- Индексы по алфавиту и по модулям

## Обновление документации

После изменения исходного кода:
- **Локально:** просто запустите `doxygen Doxyfile` снова
- **На GitHub:** документация автоматически обновится при пуше в `main`/`master` ветку

## GitHub Actions

Документация автоматически генерируется и публикуется через GitHub Actions workflow:
- Файл: `.github/workflows/docs.yml`
- Запускается при изменении файлов в `include/` или `Doxyfile`
- Публикует документацию на GitHub Pages
