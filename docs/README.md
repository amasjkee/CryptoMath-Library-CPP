# Документация CryptoMath Library

## Генерация документации

Для генерации документации используйте Doxygen:

```bash
doxygen Doxyfile
```

Документация будет создана в директории `docs/html/`.

## Просмотр документации

Откройте файл `docs/html/index.html` в браузере:

```bash
# Linux
xdg-open docs/html/index.html

# Или просто
firefox docs/html/index.html
# или
google-chrome docs/html/index.html
```

## Что включено в документацию

- Все классы и структуры
- Все функции и методы
- Концепты C++20
- Математические формулы (через MathJax)
- Иерархия классов
- Индексы по алфавиту и по модулям

## Обновление документации

После изменения исходного кода просто запустите `doxygen Doxyfile` снова для обновления документации.
